# kSari
## A Digital Twin Distributed Ledger
A solution which leverages the power of Blockchains to provide a hassle free and secure protocol for maintaining and sharing ownership. 

## Tech Stack

+ VueJS
+ CSS3
+ NuCypher
+ Torus
+ Web3
+ Ethereum Blockchain
